TGA CSV Exporter

A small utility to convert raw TGA output files into clean CSV files. Designed for transparency and easy integration into existing workflows.

What this tool does

  • Reads raw TGA text output
  • Extracts numerical measurement data
  • Exports clean, analysis-ready CSV files

What this tool does not do

  • No automatic interpretation
  • No peak detection or fitting
  • No modification of original data

Input

  • Raw TGA output files (text format)

Output

  • CSV file with extracted numerical data

Download
