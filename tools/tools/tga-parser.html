TGA CSV Exporter

This tool converts raw TGA output files into structured CSV. It is designed to be simple, transparent, and easy to integrate into existing workflows.

What this tool does

  • Reads raw TGA text output
  • Extracts numerical data
  • Exports clean CSV files

What this tool does NOT do

  • No automatic interpretation
  • No fitting or modeling
  • No black-box processing
